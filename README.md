#This system convert a gif image into a video using NodeJS
====================
##install:
  - `npm install`
  - `bower install`
  - `sudo apt-get install ffmpeg`
  - `sudo apt-get install imagemagick`
##start server and run app:
  - `gulp`<br>
###ports:
  - server: localhost:3000
  - WebApp: localhost:8001
